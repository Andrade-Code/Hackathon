Cursos de programação web na modalidade à distância

Modalidade de oferta:

Os cursos serão desenvolvidos pelo Ifes campus Cachoeiro de Itapemirim utilizando-se de Ambientes Virtuais de Aprendizagem, contando com orientação de profissionais capacitados como professores formadores, mediadores e monitores.

Os estudos ocorrerão online, com apoio e orientação presencial no polo de apoio presencial de seu município.

Duração:

O curso tem duração de 8 meses, divididos em 4 meses para Programação básica de aplicativos Web - Front-end e 4 meses para Programação com JavaScript e o Framework Node.js - Back-end.

O curso tem início em setembro de 2022 e término previsto para maio de 2023.

Carga horária semanal prevista:

O estudante deve dedicar pelo menos 10 horas semanais para estudos.

Confira a grade curricular do curso:

Curso FIC 1 - Programação básica de aplicativos Web - Front-end
Módulo / Período Conteúdos Carga Horária
Módulo I: Conhecimentos básicos de HTML e CSS
  • Funcionamento de elementos de um site: HTML, CSS, JavaScript e imagens.
  • Elementos para criação e formatação de tabelas e formulários.
  • Agrupamento, dimensionamento e posicionamento de elementos.
 40 horas
Módulo II: Introdução ao framework Bootstrap
  • Introdução ao Bootstrap 5.
  • Formulários e componentes do framework.
  • Classes utilitárias para formatação de páginas.
 40 horas
Módulo III: Programação estruturada com JavaScript
  • Introdução à linguagem JavaScript.
  • Estruturas condicionais e introdução à coleções de dados.
  • Estruturas de repetição e coleções de dados.
  • Funções e procedimentos.
 50 horas
Módulo IV: Desenvolvimento web com HTML, CSS e JavaScript
  • Hierarquia e propriedades de objetos de um documento HTML.
  • Manipulação de objetos de um documento HTML.
  • Eventos de objetos de um documento HTML.
  • Validação de formulários.
 30 horas
Carga horária total 160 horas
Curso FIC 2 - Programação com JavaScript e o Framework Node.js - Back-end
Módulo / Período Conteúdos Carga Horária
Módulo I: Introdução ao framework Node.js e notação JavaScript
  • Instalação, configuração e introdução ao framework Node.js.
  • JavaScript com Node.js e introdução à notação de objetos JavaScript (JSON).
  • Depuração de programas e programação assíncrona.
  • Requisições HTTP.
 40 horas
Módulo II: Criação de páginas customizadas com Node.js
  • Servidor web com Node.js.
  • Criação de páginas customizadas com templates avançados.
  • Manipulação de arquivos e envio de mensagens.
  • Envio e formatação de arquivos e imagens.
 40 horas
Módulo III: Persistência de dados com Sequelize
  • Introdução ao Sequelize.
  • Persistência de dados.
  • Adicionar e remover objetos com Sequelize.
  • Alterar e realizar consulta de objetos com Sequelize.
  • Consulta e filtro de objetos com Sequelize.
 40 horas
Módulo IV: Node.js intermediário, CRUD com Sequelize, cookies e sessões
  • Funcionamento de rotas e parâmetros de rotas.
  • Criação de CRUD usando Sequelize.
  • Introdução e manipulação de cookies e sessões.
  • Autenticação de usuários.
 40 horas
Carga horária total 160 horas
